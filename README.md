# c_practice_by_zhs

这个仓库用于记录我在学习 **C 语言** 过程中的练习代码、示例和小项目。  
主要目标：

- 熟悉 C 语言的语法和常用标准库函数
- 通过有针对性的练习掌握数组、指针、结构体、文件操作等核心知识
- 最终能够独立完成一些小项目（如简单游戏、工具程序等）

---

## 📁 目录结构

当前仓库的目录大致如下：

```text
.
├── basics/      # 基础语法：变量、数据类型、运算符、分支、循环、函数
├── arrays/      # 一维数组、多维数组、字符串相关练习
├── pointers/    # 指针基础、指针与数组、指针与函数、指针与结构体
├── structs/     # 结构体、共用体、枚举以及相关练习
├── files/       # 文件读写：fopen/fclose、fprintf/fscanf、二进制读写等
└── projects/    # 综合小项目（比如通讯录、学生成绩管理、小型游戏等）
```

> 后续随着学习深入，目录可能会根据需要调整和扩展。

## 🔰 学习计划（草案）

1. **基础部分（basics）**
   - C 程序的结构与编译运行流程
   - 变量与基本数据类型（int / float / char / double 等）
   - 运算符与表达式
   - 分支语句：`if` / `switch`
   - 循环语句：`for` / `while` / `do...while`
   - 函数的定义与调用、参数传递、作用域

2. **数组与字符串（arrays）**
   - 一维数组、二维数组
   - 字符数组与 C 风格字符串
   - 常用字符串处理函数：`strlen` / `strcpy` / `strcmp` / `strcat` 等

3. **指针（pointers）**
   - 指针变量与内存地址的概念
   - 指针与数组、指针与函数
   - 指针与结构体
   - 动态内存分配：`malloc` / `calloc` / `realloc` / `free`

4. **结构体与自定义类型（structs）**
   - 结构体定义与使用
   - 结构体数组、结构体指针
   - 枚举类型 `enum` 与共用体 `union`

5. **文件操作（files）**
   - 文本文件读写：`fopen` / `fclose` / `fprintf` / `fscanf` / `fgets` / `fputs`
   - 二进制文件读写：`fread` / `fwrite`
   - 简单的数据持久化练习

6. **综合小项目（projects）**
   - 学生成绩管理系统
   - 通讯录管理
   - 简单小游戏（如数字猜谜、扫雷简化版等）
   - 根据学习情况逐步增加

## 💻 编译与运行（示例）

在本地环境中，如果安装了 `gcc`，可以通过命令行编译运行示例程序，例如：

```bash
# 进入 basics 目录
cd basics

# 编译 hello_world.c
gcc hello_world.c -o hello_world

# 运行
./hello_world
```

不同文件会有各自的编译命令，后续可以根据需要编写简单的脚本或 Makefile 来统一管理。
